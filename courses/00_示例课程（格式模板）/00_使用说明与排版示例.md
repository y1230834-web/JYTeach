# 示例课程 · 格式模板

> 这是一份「讲解型」文档示例：纯阅读、自动排版。
> 同时它也是新课程的写作模板——把你的 `.md` 按本课程的几份文件依葫芦画瓢即可。

## 一、这门网站能做什么

1. **多课程切换**：左上角下拉框切换不同课程，内容整体替换。
2. **分级导航**：左侧点开任一文档，会在其下展开 `##` / `###` / `####` 各级小标题；
   点击小标题，右侧内容自动滚动跳转，并随阅读位置高亮当前小节。
3. **讲解型 .md**：表格、列表、引用、`代码` 等自动渲染。
4. **做题型 .md**：判断 / 选择 / 填空 / 简答等，前端可交互作答、看答案、看解析。
5. **错题集**：任意题卡右上角点「☆ 加入错题集」即可收集；左侧「错题集 → 错题训练（乱序）」
   每次进入都会随机打乱出题，「错题管理」里可逐题删除或一键清空。
   错题集保存在你浏览器本地（localStorage），部署到公网后照常保存与修改——
   但它**按设备 / 浏览器各自保存，不跨设备同步**。

## 二、排版能力速览

### 2.1 表格

| 能力 | 说明 | 支持 |
|---|---|---|
| 表格 | GFM 表格 | ✅ |
| 列表 | 有序 / 无序 | ✅ |
| 引用 | `>` 块引用 | ✅ |

### 2.2 列表与强调

- 支持 **加粗**、*斜体*、`行内代码`。
- 嵌套列表：
  - 子项一
  - 子项二

### 2.3 引用

> 「把复习资料写成 Markdown，让网站替你排版和出题。」

## 三、目录结构约定

```

├── 00_示例课程（格式模板）/   ← 本课程
│   ├── 00_使用说明与排版示例.md

├── 政治学原理/                 ← 另一门课
│   └── *.md
└── 你的新课程/                 ← 以后新增：放一个子文件夹即可
    └── *.md
```
